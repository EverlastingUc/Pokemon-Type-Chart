## Overview

Pokémon Type Chart adds a type effectiveness chart overlay on Pokémon Showdown. Quickly see which types are super effective, not very effective, or have no effect against any Pokémon type.

---

## Features

- Full type effectiveness chart for Generation 9.
- 2× (Super Effective), ½ (Not Very Effective), 0 (No Effect), 1 (Normal).
- Dark and light mode toggle.
- Help section explaining how to read the chart.
- Click outside or press Escape to close.
- Toggle button to show/hide the chart.

---

## Installation

1. Install Tampermonkey or Violentmonkey
2. Click [here](https://raw.githubusercontent.com/EverlastingUc/Pokemon-Type-Chart/main/typechart.user.js) to install
3. Visit Pokémon Showdown
4. Click the "Type Chart" button in the bottom-right corner

---

## How To Use

Click the **Type Chart** button in the bottom-right corner to open the chart. Use the **?** button for help, the **🌙/☀️** button to toggle dark/light mode, and the **✕** button to close.
