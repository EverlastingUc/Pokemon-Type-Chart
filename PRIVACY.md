# Privacy Policy

**Last Updated:** 28 June 2026

## 1. Data Collection

This script collects zero data. No personal information, analytics, cookies, or IP logging occurs. No data of any kind is ever sent to any server.

- No personal data is requested or received
- No analytics or telemetry is collected
- No cookies are created or accessed
- No IP addresses are logged
- No third-party data collection tools are integrated

---

## 2. Local Storage Usage

This script does not use localStorage. No settings are saved.

---

## 3. Data Storage

No data is transmitted, shared, or stored on any external servers.

- No cloud storage
- No remote databases
- No third-party APIs
- No analytics services

---

## 4. Security

All operations performed by this script occur entirely within your local browser environment. No network requests are made. The script has no external dependencies and does not communicate with any remote servers.

- All processing is done locally
- No network requests are initiated
- No remote code execution
- Your privacy is fully maintained

---
